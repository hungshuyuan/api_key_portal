1. 先安裝vscode
2. 安裝vscode的github copilot外掛
3. 安裝vscode的LiteLLM Provider for GitHub Copilot Chat外掛
4. 按下Ctrl+Shift+P，輸入「Manage LiteLLM Provider」，輸入‵http://163.18.26.230:7536/`和你的API Key，按下Enter
5. 按下Ctrl+Shift+P，輸入「Manage Language Models」，你會在下方看到LiteLLM群組，將滑鼠移至該模型例如「gpt-oss-120b」，按下左側的眼睛，其他模型同理，這樣就完成設定了
6. 按下Ctrl+Shift+I，就可以開啟Copilot Chat視窗，在下方選擇模型的地方就可以看到你剛剛設定的模型了，選擇後就可以開始使用了
